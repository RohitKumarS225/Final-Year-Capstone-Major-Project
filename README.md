# Final-Year-Capstone-Major-Project
Developed a YOLOv11-based system to detect and segment handwritten Telugu characters from images, videos, and webcam feeds. Built using Python, Anaconda, and Label Studio for dataset annotation, supporting Telugu OCR and document digitization.

I have built this project based on EDJE Electronics' explanation.
